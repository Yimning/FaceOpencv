1、需要注意安装opencv的路径，在下载安装时需要注意把代码中的路径改掉。
2、若出现“no opencv_java430 in java.library.path”的错误，

3、使用的软件为idea。
4、opencv安装包(win64)。
5、导入工程时需要把项目的lib下的包用上，否则报错。
